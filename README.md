# 💰 Expense Tracker (Python)

## 📌 Project Overview
This is a simple Command Line Interface (CLI) Expense Tracker built using Python. It helps users record daily expenses, view all expenses, and calculate the total amount spent.

## ✨ Features
- ➕ Add a new expense
- 📋 View all expenses
- 💵 Calculate total expenses
- 📅 Store date, category, description, and amount
- ❌ Exit the application

## 🛠️ Technologies Used
- Python 3
- Lists
- Dictionaries
- Loops
- Conditional Statements

## ▶️ How to Run

```bash
python main.py
```

## 📷 Sample Menu

```
1. Add Expense
2. View All Expenses
3. View Total Expense
4. Exit
```
